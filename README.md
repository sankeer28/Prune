## Requirements

- [Node.js](https://nodejs.org) 18+
- [Ollama](https://ollama.com) running locally
- iTunes or Apple Mobile Device Support (for iPhone import on Windows)

## Setup

```bash
npm install
npm run dev
```


## Build

```bash
npm run build
```
Produces a Windows installer in `dist/`.
